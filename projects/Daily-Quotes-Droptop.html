Daily Quotes is an app extension for the Droptop Four bar that displays Daily Quotes on your bar.

I have readjusted the Illustro Quotes skin by Eclectic-Tech to fit in the app and in the Droptop Four style, and in future updates I'm going to increse the features that this app offers.



Droptop

Droptop Four is the fourth iteration of the popular dropdown app launcher for Windows & Rainmeter. Droptop Four brings new features, plugins, themes, and improved compatibility with Windows 10.
Droptop allows you to catalog your applications into scrollable groups by taking directory, filename, and icon information directly from any specified folder, allowing easy adding and removing shortcuts, renaming, and icon support through Windows. Easily drag & drop your shortcuts into Droptop's folders to add shortcuts, or rename, delete, or change a shortcut's icon all through the standard Windows interface. Feature and option-rich settings menu allow you full control over the appearance and functionality of your dropdown menus.

Download


To use the Daily Quotes app you need to install Droptop Four.
There are 2 ways to install this Rainmeter package:

  • The first way is downloading the *.rmskin* package from the GitHub releases or from my DeviantArt page. Opening the file will open the installer and, after making shure that the option *Load included skins* is ticked to open the readme document with the guide, you can click on install.
  • The second way is cloning this repository in the Droptop Community Apps directory "\Rainmeter\Skins\Droptop Community Apps"
